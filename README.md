## ⚙️ Installation steps
Follow these steps to set up the software on your Windows 11 computer.

1. Double-click the downloaded file. Windows will ask for permission to change your system. Select Yes.
2. The installation wizard will open. Click Next to proceed through the introduction screens.
3. Choose the default folder for installation. For most users, the standard C:\Program Files\KiCad path works best.
4. Select the components you wish to install. If you remain unsure, keep all boxes checked. This ensures you receive the full set of libraries and 3D shapes.
5. Click Install. Wait for the progress bar to finish. This process takes a few minutes depending on your computer speed.
6. Click Finish to close the wizard.

## 🚩 Fixing common errors
If the installation stops or returns an error, try these fixes.

**Administrator errors**
If the installer fails, right-click the setup file and select "Run as administrator." This gives the program the access it needs to write files to your protected system folders.

**Missing libraries**
At times, the installation finishes but the software lacks footprints or symbols. Open the KiCad application. Go to Preferences, then Manage Symbol Libraries. Click the small folder icon to add defaults. Repeat this for footprint libraries if needed.

**Security warnings**
Windows may flag the installer as unrecognized. Click "More info" in the pop-up box, then select "Run anyway." This confirms you trust the source.

**Corrupted download**
If the installation stalls, your file might be incomplete. Delete the downloaded file, clear your browser cache, and download the installer again from the link provided above.

## 📋 Best practices for setup
Keep your software updated. KiCad releases patches to fix bugs and add features. Periodically check the website to see if a newer version exists. Save your work designs in a dedicated project folder outside of the installation directory. This protects your files if you ever need to reinstall the software.

## 💡 Troubleshooting tips

Ensure you have the latest updates from the Windows Update settings menu. If you see a specific error code, search that code in your browser.
